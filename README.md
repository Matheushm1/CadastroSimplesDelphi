Cadastro Simples de Cliente – Delphi

Este projeto foi desenvolvido utilizando Delphi com o objetivo de praticar e demonstrar conhecimentos básicos na linguagem, focando principalmente em lógica de programação, manipulação de componentes visuais e validação de dados.

O sistema consiste em um cadastro simples de pessoas, permitindo registrar informações básicas e organizar os dados em listas diferentes de acordo com o tipo de usuário.

Funcionalidades

* Cadastro de nome e endereço

* Seleção do tipo de pessoa:

* Pessoa Física

* Pessoa Jurídica

* Seleção do tipo de cadastro:

* Cliente

* Fornecedor

* Vendedor

* Validação de campos obrigatórios

* Exibição dos dados cadastrados em listas separadas

Este projeto foi criado com foco em praticar:

* Estruturas condicionais (if / else)

* Estrutura case

* Manipulação de componentes VCL

* Validação de campos de formulário

* Manipulação de strings

* Organização básica de lógica em Delphi

Componentes utilizados

* TEdit

* TRadioGroup

* TComboBox

* TMemo

* TButton

Funcionamento

O usuário preenche os campos de nome e endereço, escolhe se a pessoa é física ou jurídica e seleciona se o cadastro será para cliente ou vendedor.

Após isso, o sistema registra os dados na lista correspondente dentro da aplicação.

Objetivo do projeto

Este projeto foi desenvolvido com finalidade educacional, visando praticar os fundamentos da programação em Delphi antes da implementação de recursos mais avançados como:

Integração com banco de dados,
Persistência de dados,
Estruturação em camadas

Tecnologias utilizadas

* Delphi (Embarcadero RAD Studio)

* VCL (Visual Component Library)
